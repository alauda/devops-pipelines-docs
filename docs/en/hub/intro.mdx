# Introduction

Tekton Hub is a platform for developers to discover, share, and contribute Tekton tasks and pipelines.

## Key Features

- Tekton Hub provides the following features:

  1. Display resources in a curated way:

     Users can get detailed information such as descriptions, YAML, tags, and categories.

  2. Filter by category:

     This allows users to filter resources by category, which helps them access multiple resources.

  3. Search for resources

  4. Centralized task and pipeline management
